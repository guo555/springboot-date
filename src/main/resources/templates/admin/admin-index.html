华中师范大学计算机学院会议室管理办法与规则

会议室是计算机学院重要的学术交流活动场所，为加强会议室的管理，做到合理使用，提高会议室利用率，保证实验室的各类活动顺利开展，制定本办法。

会议室主要用于实验室的各类工作会议、学术报告、课题组组会、研究生答辩、接待检查和来访等活动。如有特殊情况，需递交纸质申请经办公室同意方可使用。

一、会议室的管理与维护

1、实验室办公室负责会议室锁匙管理，会议室使用审批。

2、实验室办公室负责安排物业管理人员做好会议室设备检修及会议室清洁卫生工作，保证会议室整洁卫生，设施完好。

二、会议室的使用要求

1、会议室的使用遵循“谁使用，谁负责”的原则。

2、会议室的使用采用预约制，使用者提前通过实验室网站进行预约，按实际参加人数预约，不得无故占用大会议室，非预约不得使用。已经审批通过的各类活动如与实验室临时安排的重要活动冲突，则服从实验室统筹安排。

3、使用者对活动内容和活动安全负责，禁止散布与党的路线方针政策不符、影响安定团结的言论。

4、会标、欢迎牌等宣传物品的使用需经实验室办公室同意，不得破坏、弄脏原有装修，会后应及时清理。不得随意从其他会议室搬动桌椅，不得随意改变会议室设备和家具的位置，如需改变，应提前与办公室说明，在使用结束后主动恢复原状。

5、会议室的照明、空调、音响设备的使用原则上由使用者负责，办公室协助指导使用。会议室使用完毕后，使用者应及时整理，关闭电源、门窗，室内保持整洁。如发现设备故障和公物损坏应及时告知办公室，以便及时修理。因使用者的原因导致设备损坏或丢失的，由使用者负责维修或照价赔偿。

6、禁止在会议室内抽烟、喝酒，禁止使用易燃、易爆及有毒有害物品。

7、对不遵守会议室管理办法的使用者，由实验室予以通报批评，情节严重的，取消其今后使用实验室会议室的资格，并对相关责任人追究相应责任。
